Electric Dough — Рок-пекарня
Electric Dough — это веб-приложение для тематической пекарни, объединяющей рок-культуру и домашнюю выпечку. 
Проект разработан в рамках дипломной работы и предназначен для автоматизации процессов бронирования столиков, 
оформления предзаказов и управления контентом заведения.

📋 Функциональные возможности
 Для посетителей (гостей):
- Просмотр каталога товаров с фильтрацией по категориям (выпечка, десерты, напитки)
- Генерация случайных цитат рок-звёзд через внешний API
- Игра «Рандомайзер блюд» — случайный выбор товара из меню
- Регистрация и авторизация с подтверждением email
 Для авторизованных пользователей:
- Личный кабинет с редактированием данных
- Бронирование столиков с визуальным выбором места через схему зала
- Оформление предзаказа продукции с выбором даты и времени самовывоза
- История бронирований и предзаказов
- Программа лояльности (уровни «Новичок», «Звезда», «Легенда»)
- Отмена активных бронирований
- Форма обратной связи
Для администратора:
- Админ-панель для управления сайтом
- Управление товарами (добавление, редактирование, удаление)
- Управление бронированиями (просмотр, подтверждение, отмена)
- Управление предзаказами (изменение статуса)
- Управление пользователями
- Просмотр сообщений из обратной связи


🚀 Установка и запуск
Требования
- Open Server Panel (или другой локальный сервер с PHP 8.1+ и MySQL 8.0+)
- Браузер (Chrome, Yandex, Firefox)
Установка:
1. Скопируйте проект в папку OSPanel/domains/
2. Запустите Open Server Panel
3. Импортируйте БД:
  - Откройте phpMyAdmin
  - Создайте базу rock-bakery
  -Импортируйте файл sql/rock-bakery.sql
4. Откройте в браузере: http://Electric-Dough
